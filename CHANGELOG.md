# Changelog

All notable changes to AutoTune will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-12-01

### Added
- Initial release
- 30-phase exhaustive hardware benchmark
- Per-setting continuous optimization (no presets)
- Live adaptive performance engine with state machine
- 8-screen interactive questionnaire
- Full Mod Menu integration with 7-tab UI
- Profile system with import/export/sharing
- Sodium and Iris compatibility and optimization
- GPU database (300+ models) and CPU database (200+ models)
- Contextual awareness (Nether, End, elytra, mob farms)
- Toast notification system for live adjustments
- Benchmark report generation
- Support for Minecraft 1.21 through 1.21.11
- Fabric and Quilt loader support
